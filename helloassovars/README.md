# HelloAssoVars FOR [DOLIBARR ERP CRM](https://www.dolibarr.org)

## Features

Simply add a missing feature in [HelloAsso module](https://www.dolistore.com/product.php?id=2332&title=helloasso&l=fr) to allow the use of email template with `__HELLOASSO_PAYMENT_URL__` substitution.
